UPI Transaction Performance & Reliability Analysis
Overview

This project analyzes 250K+ UPI transactions to evaluate system reliability and scalability under varying transaction loads.

Objective

Measure overall failure rate

Analyze peak-hour load impact

Identify device-level risk

Evaluate monthly stability

Tech Stack

Python (Pandas, NumPy)

Tableau

Jupyter Notebook

Key Insights

Failure rate remains stable at ~5%

No strong correlation between peak load and failures

Web channel shows slightly higher failure rate

System performance stable across months
